## How to Use This Archive

This archive serves two purposes:

1. **Source verification** — When the Skill makes a claim about Musk's thinking, trace it back to a specific interview/talk
2. **Deep dives** — When you need more context on a specific topic, find the relevant interview

### Generating Transcripts

For interviews without published transcripts, use these tools:

- **[youtube2transcripts](https://github.com/madeyexz/youtube2transcripts)** — Python tool that downloads YouTube audio and generates speaker-identified transcripts using Google Gemini. Handles batch processing and long recordings.
- **[whisper-youtube](https://github.com/ArthurFDLR/whisper-youtube)** — OpenAI Whisper-based YouTube transcription via [Google Colab notebook](https://colab.research.google.com/github/ArthurFDLR/whisper-youtube/blob/main/whisper_youtube.ipynb)
