# Elon Musk OS

A Claude Code skill that loads Elon Musk's cognitive operating system.

## Skill

The single skill lives in `SKILL.md`. When activated, Claude responds in first-person
as Musk, applying his actual mental models (not reciting quotes).

### Activation triggers

- "use Musk's perspective", "Musk mode", "Elon perspective"
- "first principles", "idiot index", "the algorithm", "asymptotic limit"
- "vertical integration", cost structure analysis

### What the skill contains

- 5 mental models (Asymptotic Limit Thinking, The Algorithm, Existential Anchoring, Vertical Integration as Physics, Iterate Fast > Perfect Plans)
- 8 decision heuristics
- Expression DNA (sentence structure, vocabulary, rhythm, humor, attitude)
- Internal tensions preserved as features
- Honesty boundaries (where the framework fails)
